# First Person Player For Unity

A First Person Player for RPG Game made by help of [Brackeys](https://www.youtube.com/c/Brackeys)'s YT tutorials.

- [x] Locomotion
- [x] Jump
- [x] Head Bobbing
- [x] Four Weapon Attached
- [x] Switching Weapon with keypad and mouse
- [x] Mini Map
- [x] A Demo Scene
- [x] Walking Audio 
- [x] Camera Shake
